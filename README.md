# onscreen-keyborad
A Python onScreen Keybord


# Short cuts

| Action                        | Short cut    |
| ----------------------------- | ------------ |
| Back                          | Ctrl + Z     |
| Repeat                        | Ctrl + Y     |
| Increase/decrease brush width | ] [          |
| Pippete (brush)               | alt          |
| Size                          | Up           |
| Size                          | Down         |
| Size  ~ 10                    | Shift + Up   |
| Size  ~ 10                    | Shift + Down |
| Brush                         | *            |
| Erase                         | -            |
| Pippete                       | /            |
